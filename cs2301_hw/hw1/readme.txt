RUNNING THE PROGRAMS:

To run these program, unzip all files, and navigate to the directory containing
them within a UNIX terminal. To compile the code, enter

$make

Following compilation, to run the leap year program, enter:

$./leap <year>

with <year> being replaced with a numeric year equal to or more recent than
1752. 

To run the grade program, enter: 

$./num [grades]

with [grades] being replaced with one or more numeric integer grades.
